### 起源
社区起名源于我们一个有好几个小伙伴的群 PPAP，人数为 6 ，故命为 PPAP6。  

PPAP6 组织的建立由技术驱动，以做项目经历积累技术经验为目标。  

👻 热血！从不会感到疲倦！

如果你觉得有趣或者对你有帮助，可以点个 star 支持一下！
